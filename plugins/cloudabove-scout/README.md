## Cloud above scout

### About
This plugin allows you to purge the Scout cache from the WordPress admin 

### Requirements
* PHP 7.2.5
* Composer

### Installation
1. Clone the repository
2. Run `composer install`.

### Built with
* [Guzzle](https://docs.guzzlephp.org/en/stable/)
